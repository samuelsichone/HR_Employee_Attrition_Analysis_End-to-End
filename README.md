HR Employee Attrition Analysis

Overview:
End-to-end analytics project focused on understanding employee turnover and identifying key drivers of attrition.

Technologies:

Python (data cleaning, analysis, visualization)

SQL (data extraction & querying)

Power BI (interactive dashboards & KPI reporting)

Key Features:

Cleaned and preprocessed HR datasets for accurate analysis

Conducted exploratory data analysis to uncover trends and patterns

Built interactive dashboards highlighting attrition metrics and KPIs

Provided actionable insights to inform HR strategies and reduce turnover

Outcome:
Delivered data-driven recommendations to improve employee retention and workforce planning
